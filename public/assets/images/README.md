### Hero Section
- `banner.jpg` - Hero background image (recommended: 1920x1080px, high quality)

### Gallery Images (Before/After pairs)
- `before-1.jpg` through `before-4.jpg` - Before photos for gallery
- `after-1.jpg` through `after-4.jpg` - After photos for gallery
- Recommended size: 800x600px, high quality

### General

## File Format Guidelines:
- Use `.png` for logos (supports transparency)
- Use `.jpg` for photos (smaller file size)
- Optimize images for web (compress without losing quality)
- Use descriptive filenames for better SEO

## Adding New Images:
1. Place image files in this `/public/assets/images/` folder
2. Reference them in components using `/assets/images/filename.ext`
3. Always provide alt text for accessibility
